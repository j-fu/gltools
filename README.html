OpenGL Graphics with gltools

OpenGL provides a simple to understand, portable API for low level 3D graphics programming. The gltools package gives easy access to these possibilities, and is designed especially for developers of numerical codes. For more information why we did this package, see the documentation below and the gltools homepage.
